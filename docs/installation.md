# Installation

## Windows 🪟

To install this app on Windows:

1. [Download](https://github.com/yoptohlejepeta/marek/releases/latest/download/MAReK-Windows-Setup.exe)
2. Extract downloaded `MAReK-Windows-Installer.zip`.
3. Proceed to install through the installation dialog.
4. After installation, icon should appear on your desktop.

::: info
During installation, `uninstall.exe` appears in the destination directory.
:::
